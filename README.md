# Maria_Hernandez_MiWeb
Actividad evaluable de Lenguaje de Marcas, 1º trimestre.
Esta activida consistirá en diseñar y desarrollar un sitio web estático de múltiples páginas, demostrando la aplicación
integral de los estándares HTML5 (estructura y contenido) y CSS3 (presentación y
maquetación). El proyecto debe evidenciar una estructura de código profesional, una
correcta implementación semántica y el dominio de técnicas de maquetación modernas
